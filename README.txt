Applicataion: Asp.net core web API 
Database: sql server (DB name: CodeTestDB), initial local db using script.sql in the root path, db connection will be windows authentication
APIs: 
GetCustomers: List all customers
GetCustomer: Get a customer by customer id
PutCustomer: Create a new customer 
PostAccount: Create a new account for customer
PostAmount: Save money into customer account
TransferAmount: Transfer money between customer accounts
